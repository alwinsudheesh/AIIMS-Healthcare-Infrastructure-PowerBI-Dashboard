# Healthcare Infrastructure Analytics of AIIMS Hospitals in India

This project presents a Power BI dashboard analyzing healthcare infrastructure across AIIMS hospitals in India.

## Dashboard Features

- Total Oxygen Beds
- Total ICU Beds
- Total Operational Beds
- Hospital infrastructure comparison
- ICU vs Oxygen bed distribution
- Interactive hospital filter

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Visualization

## Dataset

Healthcare infrastructure data of AIIMS hospitals including:
- Non-ICU Oxygen Beds
- ICU Beds
- Operational Beds
